Setup Guide

After you unzip the archive, you'll find the necessary JS files in the js folder. Just include the minified version in the page in which you are already running the Fancy Product Designer.

<script src="js/FPDPricingRules.min.js" type="text/javascript"></script>

To generate the pricing rules for your product designer, we offer a handy online tool that will create the JS code that you can easily copy & paste into your website. Just go to http://fancyproductdesigner.com/addon-pricing-rules/ and scroll down to "Create Pricing Rules".

Here you can define own pricing groups and in every pricing group you can set the pricing rules for various element properties. When you are done, click on the "GET JS CODE" button and copy the pricingRules property that you need to merge with the other plugin options that you are using for the FancyProductDesigner class (See Call the plugin section).

If you still have problems after reading the documentation, there is a helpful support center with lots of solution articles, discussions and to contact the support team.
